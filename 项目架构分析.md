# 个人简历/作品集网站项目架构分析

## 技术栈

| 层级 | 技术 |
|------|------|
| 框架层 | Vue 3.5 + Composition API (`<script setup>`) |
| 构建工具 | Vite 6.0 |
| CSS方案 | Tailwind CSS v4 + 自定义全局样式 |
| 状态管理 | Pinia 2.3 |
| 路由管理 | Vue Router 4.5 |
| 动画库 | GSAP 3.12 |

## 数据驱动架构

**`config/userProfile.js` 是核心数据配置文件，数据驱动所有组件渲染。**

| 数据配置 | 渲染组件 | 说明 |
|----------|----------|------|
| `profile` | SidebarProfile | 头像、名字、签名 |
| `infoItems` | SidebarProfile | 个人信息项展示 |
| `socialLinks` | SidebarProfile | 社交链接图标 |
| `heroStats` | HeroWelcome | Hero区统计数据 |
| `abilities` | AbilityService | 技能能力卡片 |
| `cases` | CaseCarousel | 精选案例轮播 |
| `timelineItems` | TimeLineTrack | 经历时间线 |
| `githubRepos` | FilterableBlog | GitHub仓库列表 |
| `projectGridItems` | ProjectGrid | 项目网格展示 |

## 设计特色总结

| 特性 | 说明 |
|------|------|
| **主题系统** | 暗色主题 (太空黑 #050a0f) + 亮色主题 (雾青浅绿 #ecf6ce) |
| **视觉风格** | Aurora 极光渐变背景 + 玻璃态卡片 (Glassmorphism) + 粒子 Canvas 动画 |
| **响应式设计** | 移动端 (汉堡菜单) / 桌面端 (侧栏布局) 自适应 |
| **数据驱动** | 所有展示内容通过 `userProfile.js` 配置，易于维护和修改 |
| **动画系统** | GSAP 动画库 + Vue transition 组件过渡效果 |
| **路由懒加载** | 所有页面组件动态导入 (`() => import(...)`)，优化首屏性能 |
| **CSS变量** | 通过 CSS 变量实现主题切换，无需重复定义样式 |

## 关键文件说明

### `main.js` - 应用入口

```javascript
import { createApp } from 'vue'
import { createPinia } from 'pinia'
import router from './router/index.js'
import App from './App.vue'
import './assets/styles/global.css'

const app = createApp(App)
app.use(createPinia())   // 状态管理
app.use(router)          // 路由
app.mount('#app')        // 挂载
```

### `store/global.js` - 全局状态

- `isDark`: 主题状态 (暗色/亮色)
- `isMobile`: 移动端检测 (window.innerWidth < 768)
- `mobileMenuOpen`: 移动端菜单展开状态

### `router/index.js` - 路由配置

| 路径 | 页面组件 | 说明 |
|------|----------|------|
| `/` | HomePage | 主页 |
| `/projects` | ProjectsPage | 项目展示 |
| `/experience` | ExperiencePage | 经历时间线 |
| `/blog` | BlogPage | 开源仓库 |
| `/gallery` | GalleryPage | 相册 |

### `assets/styles/global.css` - 全局样式

- CSS 变量定义 (颜色、边框、文本)
- Aurora 极光渐变背景效果
- 玻璃态卡片样式 (.glass-card)
- 自定义滚动条
- 打字机光标动画 (.cursor-blink)

## 总结

这是一个典型的现代 Vue 3 SPA (单页应用) 个人作品集网站架构：

1. **技术栈现代化**: Vue 3 + Vite + Tailwind CSS v4
2. **数据驱动设计**: 所有内容集中配置，便于维护
3. **组件化架构**: 清晰的层级划分 (layout/views/sections/ui)
4. **响应式适配**: 移动端和桌面端完整支持
5. **视觉精致**: Aurora背景 + 玻璃态卡片 + 粒子动画
